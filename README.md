#  Fonctionnalités : 
- Pouvoir choisir un meme,
- Ecrire un texte sur le meme,
- Pouvoir changer la position tu texte,
- Changer la couleur du texte,
- Changer la taille du texte,
- Activer l'ombre sur le texte,
